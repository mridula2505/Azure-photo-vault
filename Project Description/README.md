# Azure Photo Vault

## Overview

Azure Photo Vault is a cloud storage project built using Microsoft Azure Blob Storage.

## Services Used

- Azure Resource Group
- Azure Storage Account
- Azure Blob Storage

## Features

- Created a Storage Account
- Created a Blob Container
- Uploaded Photos to Azure
- Viewed Blob Properties
- Generated Secure SAS URLs

## Skills Learned

- Cloud Storage
- Azure Blob Storage
- Shared Access Signatures (SAS)
- Storage Security
- Azure Resource Management

## Architecture

Laptop
→ Azure Storage Account
→ Photos Container
→ Stored Images
→ SAS Secure Access URL
